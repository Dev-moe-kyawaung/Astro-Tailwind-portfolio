# Moe Kyaw Aung — Senior Android Developer Portfolio

A modern, fast, and premium portfolio built with **Astro** + **Tailwind CSS**, showcasing Flutter & Android apps, web projects, CI/CD automation, and more.

## ✨ Features

- **Fast & Minimal**: Astro builds ultra-fast, static sites with minimal JS.
- **Modern Design**: Expressive typography, microinteractions, and refined dark mode.
- **Fully Customizable**: Easy to update projects, skills, and contact info.
- **SEO Optimized**: Meta tags, structured data, and clean HTML.
- **CI/CD Ready**: GitHub Actions workflow for automated deployment to GitHub Pages.
